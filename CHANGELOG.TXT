
Version 1.3.0 (2011-??-??)
--------------------------
- Added support for multiple images per page. You can choose the image ID to show.
- Added inserttags for pageimage data ({{pageimage}} / {{pageimage::id}} / {{pageimage_alt}} etc.)

--------------------------
- You can now resize the image by setting width & height in the frontend module.
- Fixed an image-alt text bug (Thanks to Markus Milkereit)
- Added french translation (Thanks to Lionel Maccaud)

--------------------------
- The image size is now added to the template (img height + width). Thanks to Sebastian Leitz.
- You can now set an alt-attribute for the image and a title attribute for the link. Thanks to Oskar Villani.
